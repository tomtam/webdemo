简介
使用canvas制作画布.

实现功能:
1,任意画线:
2,文本输入
3,撤销上一步操作
4,恢复撤销的操作
5,还原为原图
6,放大,缩小
7,翻转,左90度,右90度
8,关闭保存
9,上传服务器
